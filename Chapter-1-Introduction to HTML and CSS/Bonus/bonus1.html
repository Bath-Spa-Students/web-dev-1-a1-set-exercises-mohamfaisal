HTML & CSS Quick-Reference Guide

A quick reference guide for HTML and CSS tags and properties.

HTML: Hyper Text Markup Language

HTML Headings

  • Headings <h1> to <h6> - The h1 to h6 tags are used to create headers. 
              <h1>Header 1</h1>
    
          <h2>Header 2</h2>
    
          <h3>Header 3</h3>
    
          <h4>Header 4</h4>
    
          <h5>Header 5</h5>
    
          <h6>Header 6</h6>

    • Paragraph (p)

  • <p> - The p tag is used to create paragraphs. 
                <p>This is a paragraph.</p>

    • Images (img)

  • <img> - The img tag is used to add images. 
                <img src="image.jpg" alt="Image description">

    • List Items (li)

  • <li> - The li tag is used to create list items. 
                <ul>
    
              <li>List item 1</li>
    
              <li>List item 2</li>
    
            </ul>

    • Div Elements (div)

  • <div> - The div tag is used to group and style elements. 
                <div class="container"><p>This is a paragraph inside a container.</p></div>

    • Unordered List (ul)

  • <ul> - The ul tag is used to create unordered lists 
                <ul>
    
              <li>Mohammed</li>
    
              <li>Faisal</li>
    
              <li>Feroz</li>
    
            </ul>

    • Button Tag

  • <button> - The button tag is used to create clickable buttons. 
                <button type="button">Click Me</button>

    • Input Tag

  • <input> - The input tag is to take user inputs like text, password etc. 
              <input type="text" name="name">

    • Anchor Tag

  • <a> - The anchor tag is used for creating hyperlinks. 
              <a href="https://www.example.com">Link Text</a>

    • Form Tag

  • <form> - The form tag is used for creating forms. 
              <form action="/submit" method="post"><input type="text" name="name"></form>

    • CSS: Cascading Style Sheets

    Font-Family Property

  •         
          .font-family {
            font-family: Arial, sans-serif; 
          }

    • Color Property

  •         
          .color {
            color: #172d67;
          }

    • Background Color Property

  •         
          .background-color {
            background-color: #172d67;
          }

    • Font-size Property

  •         
          .font-size {
            font-size: 16px;
          }

    • Padding Property

  •         
          .padded {
            padding: 20px;
          }

    • Margin Property

  •         
          .margin {
            margin: 20px;
          }

    • Border Property

  •         
          .bordered {
            border: 1px solid #172d67;
          }

    • Height and Width Properties

  •         
          p {
            background-color: red;
            height: 250px;
            width: 500px;
          }

    • Display Property (Flex)

  •         
          .flex-container {
            display: flex;
          }

    • Transition Property

  •       
      .transition {
        transition: background-color 0.3s ease;
      }
